## What it is
`thx` is a command-line interface for Things 3 on macOS. It lets you manage tasks from the terminal while keeping Things as your mobile-friendly task app.

## Why it exists
Many developers live in the terminal, but Things 3 is a GUI app. `thx` bridges that gap so you can add, view, and complete tasks without breaking focus.

## Who it is for
- macOS users of Things 3
- Developers who want a terminal workflow
- Anyone building scripts or automations around Things

## Key features
- View lists like Inbox, Today, Upcoming
- Add and complete tasks quickly
- JSON output for scripting and automation
- Works well with shell scripts and cron jobs

## Install
```bash
brew tap peeomid/tap
brew install thx
```
