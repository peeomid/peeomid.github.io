## What it is
`agj` monitors Codex and Claude agent sessions in iTerm2 so you do not miss permission prompts. It shows which session needs attention and can jump you to the right pane.

## Why it exists
When you run multiple AI agent sessions, it is easy to miss prompts. `agj` gives you a quick status view and sends notifications when an agent is waiting for approval.

## Who it is for
- People running multiple Codex/Claude sessions
- Anyone using iTerm2 with long-running agent tasks
- Developers who want fast context switching

## Key features
- Detects permission prompts
- Lists active agent sessions
- Sends actionable macOS notifications
- Jump to the correct iTerm2 pane
